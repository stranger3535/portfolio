Leaf Disease Detection

A web app that detects plant leaf diseases using a trained deep learning model.

This project uses Python, TensorFlow, and a Convolutional Neural Network (CNN) trained on the PlantVillage dataset. Users can upload an image of a leaf, and the model predicts the disease and suggests solutions. The web app has a clean interface with search functionality and detailed explanations for each disease. It demonstrates my skills in machine learning, web development, and data handling.

Live Object Detection Using YOLOv8

A real-time object detection system using YOLOv8 and a live webcam feed. Detects multiple objects, highlights them with bounding boxes, and displays class names and confidence scores in real time. Implemented with Python and OpenCV for webcam integration, and PyTorch for YOLOv8 model inference.
